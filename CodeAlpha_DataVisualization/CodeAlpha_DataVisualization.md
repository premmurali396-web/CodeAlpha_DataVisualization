# CodeAlpha - Data Visualization

## Task 3: Data Visualization

### Project Description
This project demonstrates data visualization using the Titanic dataset. Various charts and graphs were created using Python libraries to understand passenger information and identify patterns in the dataset.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Dataset
- Titanic Dataset (CSV)

## Visualizations Created
1. Survival Count
2. Gender Distribution
3. Passenger Class Distribution
4. Age Distribution
5. Correlation Heatmap

## Project Structure

```
CodeAlpha_DataVisualization/
│
├── dataset/

├── README.md

```

## How to Run

1. Install the required libraries:

```bash
pip install pandas matplotlib seaborn
```

2. Run the program:

```bash
py visualization.py
```

## Outcome

The project successfully visualizes the Titanic dataset using different charts and graphs, making it easier to understand passenger demographics, survival statistics, age distribution, passenger class distribution, and relationships between numerical features.
